# first part
Vytvořte tabulku s výpisem záznamů (lidé, soubory, karty, ...)
Využijte nové okno, které se otevře při nějakém signálu (stisk tlačítka, klik/dvojklik do tabulky, přes menu, ...)
Přidejte formulář pro vytvoření a/nebo editaci záznamu/nastavení.
